# 模拟盘与实盘配置指南

## 📋 概述

AI OKX Trader 支持两种交易模式:
- **模拟盘**: 使用虚拟资金测试策略,无真实风险
- **实盘**: 使用真实资金交易,盈亏真实

## 🧪 模拟盘交易 (推荐新手)

### 优势
- ✅ 零风险测试策略
- ✅ 验证AI决策逻辑
- ✅ 熟悉系统操作
- ✅ 调试和优化参数

### 配置步骤

#### 1. 创建模拟盘API Key

1. 登录欧易账户 (www.okx.com)
2. 点击 **交易** → **模拟交易**
3. 进入 **个人中心**
4. 点击 **创建模拟盘APIKey**
5. 设置权限:
   - ✅ 交易
   - ❌ 提现 (模拟盘不支持)
6. 复制以下信息:
   - API Key
   - Secret Key
   - Passphrase

#### 2. 配置 .env 文件

```bash
# OKX API - 实盘 (暂时可以留空)
OKX_API_KEY=""
OKX_SECRET_KEY=""
OKX_PASSPHRASE=""

# OKX API - 模拟盘 (填入模拟盘API Key)
OKX_SIMULATED_API_KEY="your_simulated_api_key"
OKX_SIMULATED_SECRET_KEY="your_simulated_secret_key"
OKX_SIMULATED_PASSPHRASE="your_simulated_passphrase"

# 设置为模拟盘模式
OKX_TESTNET=true
```

**优势**: 配置好两套API Key后,切换模式只需修改 `OKX_TESTNET` 的值!

#### 3. 验证配置

运行系统,检查日志:
```
OKX Client initialized (testnet=True)
```

### 注意事项

- 模拟盘账户与实盘账户互通
- 模拟盘资金是虚拟的,可以随意测试
- 部分功能不支持 (如提币、充值等)
- 建议先在模拟盘运行至少1周

## 💰 实盘交易 (真实资金)

### ⚠️ 风险提示

- 实盘交易使用真实资金
- 可能造成真实的资金损失
- **强烈建议先在模拟盘验证策略**
- 建议从小资金开始测试

### 配置步骤

#### 1. 创建实盘API Key

1. 登录 www.okx.com
2. 点击 **个人中心** → **API管理**
3. 点击 **创建API Key**
4. 设置权限:
   - ✅ 交易
   - ❌ 提现 (强烈建议禁用)
5. 设置IP白名单 (可选但推荐)
6. 复制以下信息:
   - API Key
   - Secret Key
   - Passphrase

#### 2. 配置 .env 文件

```bash
# OKX API - 实盘 (填入实盘API Key)
OKX_API_KEY="your_real_api_key"
OKX_SECRET_KEY="your_real_secret_key"
OKX_PASSPHRASE="your_real_passphrase"

# OKX API - 模拟盘 (保留之前的模拟盘配置)
OKX_SIMULATED_API_KEY="your_simulated_api_key"
OKX_SIMULATED_SECRET_KEY="your_simulated_secret_key"
OKX_SIMULATED_PASSPHRASE="your_simulated_passphrase"

# 设置为实盘模式
OKX_TESTNET=false
```

**优势**: 两套API Key都配置好后,切换只需修改 `OKX_TESTNET` 的值!

#### 3. 验证配置

运行系统,检查日志:
```
OKX Client initialized (testnet=False)
⚠️ LIVE TRADING MODE
```

### 安全建议

1. **API权限**
   - ✅ 只开启交易权限
   - ❌ 禁用提现权限
   - ✅ 设置IP白名单

2. **资金管理**
   - 从小资金开始 (建议100-1000 USDT)
   - 设置合理的风险参数
   - 定期检查账户状态

3. **监控告警**
   - 启用异常告警
   - 定期查看日志
   - 关注持仓变化

4. **密钥安全**
   - 不要将 `.env` 提交到Git
   - 定期轮换API密钥
   - 使用双因素认证

## 🔄 模式切换

### 从模拟盘切换到实盘

1. 停止当前运行的系统
2. 修改 `.env` 文件:
   ```bash
   OKX_TESTNET=false  # 改为false
   OKX_API_KEY="实盘API Key"
   OKX_SECRET_KEY="实盘Secret Key"
   OKX_PASSPHRASE="实盘Passphrase"
   ```
3. 重新启动系统

### 从实盘切换到模拟盘

1. 停止当前运行的系统
2. 修改 `.env` 文件:
   ```bash
   OKX_TESTNET=true  # 改为true
   OKX_API_KEY="模拟盘API Key"
   OKX_SECRET_KEY="模拟盘Secret Key"
   OKX_PASSPHRASE="模拟盘Passphrase"
   ```
3. 重新启动系统

## 🆘 常见问题

### Q1: 模拟盘和实盘可以同时运行吗?
A: 可以,但需要使用不同的配置文件和进程。建议分别在不同目录运行。

### Q2: 模拟盘的数据和实盘一样吗?
A: 市场数据相同,但订单成交可能略有差异。

### Q3: 如何确认当前运行在哪个模式?
A: 查看启动日志:
- `testnet=True` = 模拟盘
- `testnet=False` + `⚠️ LIVE TRADING MODE` = 实盘

### Q4: 实盘运行后发现问题怎么办?
A: 立即停止系统,手动平仓,然后切换回模拟盘调试。
