# 加密货币标的自动筛选脚本使用说明

## 📋 功能概述

基于 **1H 主周期 + 5m 择时** 的超短线交易策略，自动筛选符合条件的加密货币标的。

### 核心策略
- **交易风格**：激进求稳，快进快出，只顺势做多
- **周期规则**：1H K线判断趋势，5m K线辅助择时
- **更新频率**：5分钟刷新一次（可配置）
- **输出结果**：可交易标的白名单，附带评分与风险等级

## 📊 筛选规则详解

### 硬性筛选指标

| 筛选维度 | 阈值/规则 | 说明 |
|---------|----------|------|
| **基础流动性** | 24h成交量 ≥ 5亿 USDT<br>买卖盘口价差 ≤ 0.05% | 杜绝滑点、无法成交问题 |
| **市值安全** | 价格 ≥ 0.01 USDT | 过滤极低价空气币 |
| **波动适配性** | 1H ATR ≥ 3%<br>日内振幅 5%-20% | 有波段空间，拒绝死水/极端波动 |
| **趋势规则** | 1H周期 MA5 > MA10 > MA20<br>MACD在零轴上方 | 严格顺势，只做多头趋势 |
| **择时辅助** | 5m周期 RSI 30-70 | 规避追高/抄底风险 |
| **结构有效性** | 至少2个支撑位 + 1个压力位 | 适配关键位置交易逻辑 |

### 评分机制（满分100分）

- **流动性 (30%)**：成交量越大、价差越小，得分越高
- **波动率 (20%)**：波动率接近理想区间（8-15%）得分越高
- **趋势质量 (30%)**：MA间距越大、MACD越强，得分越高
- **结构清晰度 (20%)**：支撑压力位越多，得分越高

### 风险分级

- **≥80分**：优先交易（高概率机会）
- **60-79分**：备选（可考虑）
- **<60分**：自动剔除

## 🛠️ 常见问题

### Q1: 提示 "依赖库未安装"
**A:** 运行 `pip install okx pandas numpy ta-lib loguru`

### Q2: ta-lib 安装失败
**A:** Windows用户下载预编译包：https://www.lfd.uci.edu/~gohlke/pythonlibs/#ta-lib

### Q3: 筛选结果为空
**A:** 
1. 检查网络连接
2. 放宽筛选条件（降低 `MIN_24H_VOLUME_USDT`）
3. 查看日志中的具体过滤原因

### Q4: API请求频率限制
**A:** 增加 `API_DELAY_SECONDS` 值（如改为 0.5）

### Q5: 如何只筛选特定币种
**A:** 修改 `get_all_spot_symbols()` 方法，添加白名单过滤：

```python
def get_all_spot_symbols(self) -> List[str]:
    # 原有代码...
    
    # 添加白名单过滤
    whitelist = ['BTC', 'ETH', 'SOL', 'BNB']
    symbols = [s for s in symbols if s.split('-')[0] in whitelist]
    
    return symbols
```

## 📈 性能优化建议

### 1. 并行处理（高级）

```python
from concurrent.futures import ThreadPoolExecutor

def screen_all_symbols_parallel(self):
    all_symbols = self.fetcher.get_all_spot_symbols()
    
    with ThreadPoolExecutor(max_workers=5) as executor:
        results = list(executor.map(self._screen_single_symbol, all_symbols))
    
    return [r for r in results if r is not None]
```

### 2. 缓存K线数据

```python
import pickle
from datetime import datetime, timedelta

# 缓存K线数据（5分钟有效）
cache = {}

def get_klines_cached(symbol, timeframe, limit):
    cache_key = f"{symbol}_{timeframe}"
    
    if cache_key in cache:
        data, timestamp = cache[cache_key]
        if datetime.now() - timestamp < timedelta(minutes=5):
            return data
    
    data = self.fetcher.get_klines(symbol, timeframe, limit)
    cache[cache_key] = (data, datetime.now())
    return data
```
