# 交易记录功能说明

## 功能概述

系统已集成完整的交易记录功能,在Redis中记录每次建仓、止盈、止损的详细信息,并自动计算每轮交易的盈亏情况(不考虑手续费)。

## 记录内容

### 开仓记录
- 交易ID (自动生成,格式: `{标的}_{时间戳}`)
- 交易对
- 入场价格
- 仓位大小(USDT)
- 止损价格
- 止盈价格列表
- 开仓时间
- 状态 (open/closed)

### 平仓记录
- 出场价格
- 平仓原因 (stop_loss/tp1/tp2/manual)
- 盈亏金额(USDT)
- 盈亏百分比
- 平仓时间

## 使用方法

### 1. 查看交易历史

```python
from src.data.redis_state import RedisStateManager
from src.config.settings import settings

redis_state = RedisStateManager(settings.symbol)

# 获取最近50笔交易
trades = redis_state.get_trade_history(limit=50)

for trade in trades:
    print(f"交易ID: {trade['trade_id']}")
    print(f"入场价: {trade['entry_price']}")
    print(f"出场价: {trade['exit_price']}")
    print(f"盈亏: {trade['pnl_usdt']} USDT ({trade['pnl_pct']}%)")
```

### 2. 查看交易统计

```python
# 获取统计数据
stats = redis_state.get_trade_statistics()

print(f"总交易次数: {stats['total_trades']}")
print(f"胜率: {stats['win_rate']:.2f}%")
print(f"总盈亏: {stats['total_pnl_usdt']:.2f} USDT")
print(f"平均盈亏: {stats['avg_pnl_usdt']:.2f} USDT")
```

### 3. 运行测试脚本

```bash

```

## Redis数据结构

### 交易记录键
- `ai_trader:{标的}:trade:{交易ID}` - Hash类型,存储单笔交易详情
- `ai_trader:{标的}:trade_list` - List类型,存储交易ID列表(按时间倒序)

### 字段说明
```
trade_id: 交易ID
symbol: 交易对
action: open
entry_price: 入场价格
exit_price: 出场价格
size_usdt: 仓位大小(USDT)
stop_loss: 止损价格
take_profit: 止盈价格列表(JSON)
open_time: 开仓时间(ISO格式)
close_time: 平仓时间(ISO格式)
close_reason: 平仓原因
pnl_usdt: 盈亏金额(USDT)
pnl_pct: 盈亏百分比
status: open/closed
```

## 统计指标

- **总交易次数**: 已完成的交易总数
- **盈利次数**: 盈利交易数量
- **亏损次数**: 亏损交易数量
- **胜率**: 盈利次数 / 总交易次数 × 100%
- **总盈亏**: 所有交易的盈亏总和(USDT)
- **平均盈亏**: 总盈亏 / 总交易次数
- **最大盈利**: 单笔最大盈利金额
- **最大亏损**: 单笔最大亏损金额

## 注意事项

1. **不考虑手续费**: 当前盈亏计算不包含交易手续费
2. **数据持久化**: 所有记录永久保存在Redis中
3. **按标的隔离**: 不同交易对的记录独立存储
4. **自动记录**: 无需手动操作,开仓和平仓时自动记录
