# 手动设置止损止盈指南

## 使用场景

1. 在 OKX App 手动买入后，需要让程序接管止损止盈监控
2. 程序重启后 Redis 数据丢失，需要补设止损止盈
3. 需要调整当前持仓的止损止盈位置

## 使用脚本（推荐）

脚本会**自动从 OKX 获取持仓信息**，只需输入入场价和止损止盈价格：

```bash
python scripts/set_stop_loss.py
```

### 命令行快速模式

```bash
# 直接传入参数，跳过交互
python scripts/set_stop_loss.py --sl 41900.0 --tp 44200.0,45000.0

# 指定标的和入场价
python scripts/set_stop_loss.py --symbol ETH-USDT --entry 2800.0 --sl 2720.0 --tp 2950.0
```

## 方式2：手动编辑 JSON 文件

当 Redis 不可用时，直接编辑本地状态文件：

```bash
# 文件路径
logs/position_state.json
```

写入内容（字段名必须准确）：

```json
{
  "symbol": "BTC-USDT",
  "entry_price": 42300.0,
  "size_btc": 0.02337,
  "stop_loss_price": 41900.0,
  "take_profit_prices": [44200.0, 45000.0]
}
```

保存后**重启程序**，程序会从文件恢复持仓状态并启动价格监控线程。

## 止损止盈计算建议

根据默认策略参数（`MIN_RISK_REWARD_RATIO=1.5`）：

1. **止损空间** = 0.3–0.5 倍 1H ATR
2. **盈亏比** ≥ 1.5（止盈空间 ≥ 1.5 × 止损空间）
3. **止盈建议**：
   - TP1 = 入场价 + 1.5 × 止损空间（平 50%）
   - TP2 = 入场价 + 2.5 × 止损空间（平剩余）

## 常见问题

### Q: 如何查看当前 1H ATR？

```python
from src.data.okx_client import OKXClient
from src.indicators.ta_calculator import TACalculator

client = OKXClient()
calc = TACalculator()
klines = client.get_klines("BTC-USDT", "1H", 30)
result = calc._calculate_indicators(klines, "1H")
print(f"1H ATR: {result.get('atr')}")
```

### Q: 设置后可以修改吗？

可以。重新运行 `python scripts/set_stop_loss.py` 即可覆盖，运行中的 bot 自动热加载。

### Q: 没有 Redis 怎么办？

只使用 JSON 文件方式（方式2），程序重启后会从 `logs/position_state.json` 恢复。

## 安全提示

- ⚠️ 止损价必须**低于**当前价格
- ⚠️ 止盈价必须**高于**当前价格
- ⚠️ 盈亏比建议 ≥ 1.5
