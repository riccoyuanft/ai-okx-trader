# 回测模块使用说明

## 📖 概述

回测模块用于验证AI交易策略的历史表现，通过模拟真实交易流程，评估策略的盈利能力和风险特征。

## 🚀 快速开始

### 基础用法

```bash
python -m backtest.run \
    --start-date 2024-01-01 \
    --end-date 2024-12-31 \
    --initial-capital 10000 \
    --symbol XAUT-USDT
```

### 参数说明

| 参数 | 必需 | 默认值 | 说明 |
|------|------|--------|------|
| `--start-date` | ✅ | - | 开始日期，格式：YYYY-MM-DD |
| `--end-date` | ✅ | - | 结束日期，格式：YYYY-MM-DD |
| `--initial-capital` | ❌ | 10000 | 初始资金（USDT） |
| `--symbol` | ❌ | XAUT-USDT | 交易对 |
| `--no-cache` | ❌ | False | 不使用缓存，重新获取数据 |

## 🔧 核心功能

### 1. 数据加载与缓存

**自动缓存**：
- 首次运行从OKX API获取数据
- 后续运行使用本地缓存
- 缓存位置：`backtest/data/cache/`

**清空缓存**：
```python
from backtest.data.loader import HistoricalDataLoader
from src.data.okx_client import OKXClient

okx_client = OKXClient(...)
loader = HistoricalDataLoader(okx_client)
loader.clear_cache()
```

### 2. 交易模拟

**真实性保证**：
- ✅ 手续费：0.1%（OKX现货标准费率）
- ✅ 滑点：0.01%
- ✅ 止损/止盈触发检测
- ✅ 资金管理

**平仓逻辑**：
1. 止损触发：使用K线最低价
2. 止盈触发：使用K线最高价
3. AI信号平仓：使用收盘价

### 3. 性能指标

#### 基础指标
- 总交易次数
- 胜率
- 总收益（绝对值和百分比）
- 平均盈利/亏损

#### 风险指标
- **最大回撤**：权益曲线的最大跌幅
- **夏普比率**：风险调整后收益（年化）
- **盈利因子**：总盈利 / 总亏损

#### 交易统计
- 最大单笔盈利/亏损
- 最长连胜/连亏
- 平均持仓时间
- 平仓原因分布

## 📝 Python脚本使用

### 基础示例

```python
from src.data.okx_client import OKXClient
from src.config import settings
from backtest.engine import BacktestEngine
from backtest.analyzer import BacktestAnalyzer
from backtest.data.loader import HistoricalDataLoader

# 初始化
okx_client = OKXClient(
    api_key=settings.okx_api_key,
    secret_key=settings.okx_secret_key,
    passphrase=settings.okx_passphrase,
    testnet=settings.testnet
)

data_loader = HistoricalDataLoader(okx_client)

# 创建回测引擎
engine = BacktestEngine(
    start_date="2024-01-01",
    end_date="2024-12-31",
    initial_capital=10000,
    symbol="XAUT-USDT",
    data_loader=data_loader
)

# 执行回测
result = engine.run()

# 分析结果
analyzer = BacktestAnalyzer()
analysis = analyzer.analyze(result)

# 打印报告
analyzer.print_summary(analysis)
```

### 访问详细数据

```python
# 所有交易记录
for trade in result["trades"]:
    print(f"开仓: {trade['entry_price']}, 平仓: {trade['exit_price']}, 盈亏: {trade['pnl']}")

# 权益曲线
for point in result["equity_curve"]:
    print(f"时间: {point['timestamp']}, 权益: {point['equity']}")

# 性能指标
print(f"胜率: {analysis['win_rate']}%")
print(f"夏普比率: {analysis['sharpe_ratio']}")
print(f"最大回撤: {analysis['max_drawdown_pct']}%")
```

## ⚙️ 高级配置

### 自定义风险参数

回测使用与实盘相同的配置（`src/config/settings.py`）：

```python
# .env 文件
MAX_DAILY_RISK_PCT=3.0      # 单日最大风险
STOP_LOSS_PCT=2.0           # 止损百分比
TAKE_PROFIT_PCT=5.0         # 止盈百分比
```

### 多周期回测

```python
# 回测多个时间段
periods = [
    ("2024-01-01", "2024-03-31"),  # Q1
    ("2024-04-01", "2024-06-30"),  # Q2
    ("2024-07-01", "2024-09-30"),  # Q3
    ("2024-10-01", "2024-12-31"),  # Q4
]

for start, end in periods:
    engine = BacktestEngine(start, end, 10000, "XAUT-USDT", data_loader)
    result = engine.run()
    analysis = analyzer.analyze(result)
    print(f"{start} ~ {end}: 收益 {analysis['total_pnl_pct']:.2f}%")
```

## 🎯 注意事项

### 1. 数据要求

- **最小数据量**：需要至少120根5分钟K线（10小时）
- **数据完整性**：确保时间范围内数据连续
- **API限制**：OKX API每次最多返回100根K线，程序会自动分批获取

### 2. 执行时间

- **1个月数据**：约30秒
- **1年数据**：约3-5分钟
- **使用缓存**：第二次运行快10倍以上

### 3. 结果解读

**回测结果 ≠ 实盘结果**，原因：
- 历史数据可能存在幸存者偏差
- 市场环境会变化
- 滑点和手续费是估算值
- 未考虑极端行情（闪崩、暴涨）

**建议**：
- 在多个时间段回测
- 关注风险指标（最大回撤、夏普比率）
- 小资金实盘验证后再放大

## 🐛 常见问题

### Q1: 提示"未能加载K线数据"

**原因**：API请求失败或时间范围无数据

**解决**：
1. 检查网络连接
2. 确认OKX API配置正确
3. 缩短时间范围重试

### Q2: 回测结果为0笔交易

**原因**：AI在整个回测期间未发出开仓信号

**解决**：
1. 检查时间范围是否太短
2. 查看AI决策日志
3. 调整风险参数

### Q3: 缓存数据过期

**解决**：
```bash
# 使用 --no-cache 参数重新获取
python -m backtest.run --start-date 2024-01-01 --end-date 2024-12-31 --no-cache
```
