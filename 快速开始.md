# 快速开始指南

## 第一步：获取API密钥

### OKX API密钥
1. 登录 [OKX](https://www.okx.com)
2. 进入 **模拟盘交易** (强烈建议先用模拟盘测试)
3. 创建API密钥，权限设置为：**只读 + 交易**（禁止提现）
4. 记录下：
   - API Key
   - Secret Key
   - Passphrase

### OpenAI API密钥
1. 登录 [OpenAI Platform](https://platform.openai.com)
2. 创建API密钥
3. 确保账户有余额

## 第二步：配置项目

```bash
# 1. 克隆项目
git clone <your-repo-url>
cd ai-okx-trader

# 2. 创建虚拟环境
python -m venv venv
venv\Scripts\activate  # Windows

# 3. 安装依赖
pip install -r requirements.txt

# 4. 复制配置文件
copy .env.example .env

# 5. 编辑 .env 文件
notepad .env
```

## 第三步：填写配置

编辑 `.env` 文件，填入以下必填项：

```env
# OKX API (必填)
OKX_API_KEY=你的API密钥
OKX_SECRET_KEY=你的Secret密钥
OKX_PASSPHRASE=你的Passphrase
OKX_TESTNET=true  # 模拟盘设为true

# OpenAI (必填)
OPENAI_API_KEY=sk-你的OpenAI密钥
OPENAI_MODEL=gpt-4o

# Trading Config
SYMBOL=BTC-USDT
INITIAL_CAPITAL=1000.0
CYCLE_INTERVAL_SECONDS=300  # 5分钟
```

python -m src.main

## 第五步：监控运行

- 日志文件位于: `logs/ai_trader.log`
- AI历史记录: `logs/ai_history.json`
- 按 `Ctrl+C` 可以安全停止

## 常见问题

### 1. 提示 "Invalid API key"
- 检查 `.env` 文件中的API密钥是否正确
- 确认API密钥权限包含"交易"权限
- 如果是模拟盘，确保 `OKX_TESTNET=true`

### 2. 提示 "OpenAI API error"
- 检查OpenAI API密钥是否有效
- 确认账户有余额
- 检查网络连接

### 3. 程序一直显示 "wait"
- 这是正常的，AI会等待高概率交易机会
- 可以调整提示词让AI更激进（不推荐）
- 耐心等待市场机会

### 4. 想修改交易参数
编辑 `.env` 文件：
- `CYCLE_INTERVAL_SECONDS`: 调整循环间隔（默认300秒=5分钟）
- `MAX_DAILY_RISK_PCT`: 调整单日最大风险（默认3%）
- `INITIAL_CAPITAL`: 调整初始资金

### 5. 想修改AI策略
编辑 `src/config/prompts.py` 中的 `SYSTEM_PROMPT`
